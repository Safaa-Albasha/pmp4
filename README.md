# PMP Exam Prep — Complete Study App

A full Progressive Web App (PWA) for PMP exam preparation. Deploy to Vercel in minutes.

## 🚀 Deploy to Vercel

### Option 1: Vercel CLI
```bash
npm install -g vercel
cd pmp-exam-prep
npm install
vercel --prod
```

### Option 2: Vercel Dashboard
1. Go to vercel.com → New Project
2. Upload this folder (drag and drop the zip)
3. Vercel auto-detects Vite → click Deploy
4. Done! Your site is live.

### Option 3: GitHub + Vercel
1. Push this folder to a GitHub repo
2. Go to vercel.com → Import Git Repository
3. Select your repo → Deploy

## 🛠 Local Development
```bash
npm install
npm run dev
```

## 📦 Build
```bash
npm run build
# Output is in /dist folder
```
